# Rie (IoE) v2

This is a part remake of the Rie Discord bot from [**Island of Eternity**](https://discord.gg/eternity),
written in **Rust**.

## Feature set
- [x] Visa welcome feature
- [ ] `/anime` command for finding an anime
- [ ] `/avatar guild|user`
- [ ] Birthday feature (`/birthday info|set|toggle`)
- [ ] Genshin Impact integration (`/genshin id set|show`)
- [ ] `/google image|search`
- [ ] `/hug`, `/pat`, `/wink`
- [ ] `/manga`
- [ ] `/messagepicker`
- [ ] `/overlay`
- [ ] `/serverinfo`, `/userinfo`
- [ ] `/youtube`

Yeahh, some stuff is missing, LMAO.
